About This Project

Designed for Autonomous AI Agents and Reinforcement Learning

1. The Human Playground (This UI): This web interface lets researchers manually test the environment, understand the scoring mechanics, and establish human baselines. It wraps the core environment logic but requires no API keys to play.

2. LLM Agent Evaluation: AI agents do not see this frontend. Instead, you point your agent's code (like our inference.py script) at the internal API endpoints. The agent issues shell commands via JSON and processes the structured JSON state returned by the server.

3. Reinforcement Learning (RL): For massive scale RL training, you bypass this web server entirely. Simply import the native Python environment directly into your PyTorch/JAX training loop (from env import SysConfigEnv) for maximum speed.

API Reference

Integrate with your AI agent using these REST endpoints

POST/reset
Reset environment with a task
{"task_id": "basic_webserver"}
POST/step
Execute a configuration action
{"action": "useradd www-data --no-login"}
GET/state
Get the current server state as JSON
Returns full server state observation
GET/grade
Get reward score (0.0 — 1.0)
{"score": 0.85, "steps_taken": 7}
GET/tasks
List all available tasks
Returns 3 tasks with difficulty
GET/health
Health check endpoint
{"status": "ok", "environment": "sysconfig"}
